# Trading System Web Frontend

React + TypeScript dashboard for the Trading Simulator system.

## Quick Start

```bash
npm install
npm run dev
```

Visit http://localhost:5173

## Features

- Dashboard with runner overview
- Runner detail pages with charts
- Real-time data updates
- Responsive design

## Tech Stack

- React 18 + TypeScript
- Vite
- React Router
- TanStack Query
- Recharts
- Tailwind CSS
